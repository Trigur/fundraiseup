## Установка и запуск

### Сервер

- Добавить env файл. Пример заполнения см. в .env.example. 
- В терминале выполнить:

```bash
npm i 
npm run serve
```

### Клиент

- Добавить env файл. Пример заполнения см. в .env.example
- В терминале выполнить:

```bash
npm i
npm run build
```

- Открыть index.html из папки /public
